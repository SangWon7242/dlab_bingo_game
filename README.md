# dlab_bingo_game

4 x 4 HTML/CSS/JavaScript bingo game for GitHub Pages.

## Features

- Editable 4 x 4 bingo board
- 2, 3, or 4 bingo target selection
- Blocks game start until every cell is filled
- Limits each cell to fewer than 10 characters
- Locks editing after game start
- Shows BINGO overlay and reset button when the target is reached

## Local Use

Open `index.html` in a browser.

## Test

```bash
node --test bingoLogic.test.js
```
